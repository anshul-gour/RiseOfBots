Rise Of Bots Installation Manual

Before you install Rise Of  Bots, you need to know a few things about the program.

This program is constructed so that it can compile both in Windows and Linux. You need to first of all decide which platform you are going to use to develop your bot. We suggest that you use the Linux platform as far as possible, since the program performs far better on Linux. We have also noted some timing problems in the Windows version. In any case, if you are using this program to participate in Shaastra 2004 to be held at IIT Madras, you should note that the final event will be run on the Linux platform.

Installing Rise Of Bots

For Linux:
  1. After downloading riseofbots-x.xx.tar.gz, type tar -xvzf riseofbots-x.xx.tar.gz
  2. Go to riseofbots-x.xx/ directory and type
    ./build (This will make combat & diaplay)
  3. Compile your code & run Combat Engine.

    Click here to see a list of the various library files you will need.

Binary Requirement:
  • Xfree: You can install the rpm from the Red Hat Linux (8.0 or later) Installable Package. If you have Rat Hat (8.0 or later) or Mandrake Linux (8.0 or later) you need not perform this step.
    •
    • Mesa :
      • Mesa is downloadable from The Mesa page. Download the latest distribution in tar.gz or tar.bz2 format and extract the contents.

      Mesa-demos :

      • Mesa demos is also downloadable from The Mesa page. Download the latest version and extract the contents in same folder not overwriting.

      Use the installation manual to install mesa.

       

    • nCurses :  This library comes with most of the Linux distributions (It comes with all Red Hat distributions 8.0 onwards). If you don't have this library, try to get hold of a Red Hat CD or some other Linux distribution CD that has the nCurses rpm.


    For Windows:

    • riseofbots-x.xx-win32.zip
      1. Extract the contents of the zip file.
      2. Copy the glut32.dll( Glut ) in <windows dir>\windows\system\
      3. Compile your code & run Combat Engine.

      Click here to view the dependencies of the executables.

       

    • riseofbots-x.xx.tar.gz
      1. Extract the contents of the gz file using WinZip or any other decompression utility.
      2. Go to riseofbots-x.xx/ directory and execute
        build.bat (This will make combat.exe & display.exe)
      3. Compile your code & run Combat Engine.

     

    Library Requirement:

    • OpenGL with Glut:
      • Download glut-3.7.6-bin.zip (binary for windows) from www.pobox.com/~nate/glut.html.
      • Extract the contents of the archive and then copy the file  glut32.dll to your windows system directory (eg. c:\windows\system)
      • If you have Visual C++, copy the files glut32.lib and glut.h to the appropriate lib and include directories as given in the glut readme file.
    • PDCurses for windows
      • Download pdc25_vc_w32.zip from The PDCurses page. Follow the installation instructions present in the package.
      • You will need to copy the file curses.dll to your windows system directory (eg. c:\windows\system).

                Download pdcurs26.zip from The PDCurses page. Follow the installation instructions present in    the package.

               You will need to have the popular C compiler DJGPP installed for this.
